# README

Originally created as a PoC student project for Semantic Web seminar in Hochschule Rhein-Main (Department Computer Science), the application allows user to link publications from the dblp computer science bibliography (https://dblp.uni-trier.de) to any given lecturer. The dblp computer science bibliography provides open bibliographic information on major computer science journals and proceedings.

* Ruby version '2.6.6'; Rails Version '6.0.2.2'
* Build on haml views, styled with sass, font-awsome and uikit
* DB: sqlite3. seed file contains the lecturers names from Hochschule Rhein-Main (Department Computer Science) https://www.hs-rm.de/de/
